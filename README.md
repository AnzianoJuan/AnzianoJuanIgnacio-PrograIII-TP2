# PrograIII - TP2 - Buscador de Criptomonedas

Trabajo práctico de la materia Programación III. Aplicación que consume una API externa para listar y buscar criptomonedas.

🔗 **Demo:** [anzianojuan-progra3-tp2.vercel.app](https://anzianojuan-progra3-tp2.vercel.app)

## Descripción

Catálogo de criptomonedas construido con JavaScript vanilla (sin frameworks), que obtiene los datos desde una API externa y permite buscarlas/filtrarlas.

## Tecnologías

- JavaScript (vanilla)
- HTML
- CSS
- [Vite](https://vitejs.dev/) — bundler y entorno de desarrollo
- [pnpm](https://pnpm.io/) — gestor de paquetes
- Deploy en [Vercel](https://vercel.com/)

## Estructura del proyecto

```
├── public/          # Archivos estáticos
├── src/             # Código fuente (JS, estilos, lógica de la app)
├── index.html       # Punto de entrada
├── package.json
├── vite.config.js
└── pnpm-lock.yaml
```
